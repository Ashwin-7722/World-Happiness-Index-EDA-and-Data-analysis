📊 World Happiness Index – EDA & Data Analysis

This project performs an in-depth Exploratory Data Analysis (EDA) on the World Happiness Index dataset to uncover key factors influencing global happiness levels. Using Python and data visualization tools, the analysis identifies trends, correlations, and insights across different countries and regions.

🚀 Key Highlights

Performed data cleaning and preprocessing using Pandas

Conducted detailed EDA to analyze factors like GDP, social support, life expectancy, and freedom

Visualized insights using Matplotlib and Seaborn

Identified correlations between happiness scores and socio-economic indicators

Built interactive dashboards in Tableau for better data storytelling

🛠️ Tech Stack

Python (Pandas, NumPy)

Matplotlib & Seaborn

Tableau

📈 Outcome

Discovered key drivers of happiness across countries

Highlighted regional trends and disparities

Provided meaningful visual insights for data-driven understanding
